# Analytics Gym

How I think and what I ship.

Frameworks for growth, experimentation, and monetization. Built with Quarto, deployed on GitHub Pages.

## Live Site

[analyticsgym.github.io](https://analyticsgym.github.io)

## Adding a New Post

1. Copy a template from `_templates/` into `posts/`
2. Rename the file (use kebab-case: `my-post-title.qmd`)
3. Set `draft: false` when ready to publish
4. Push to `main` — GitHub Actions builds and deploys automatically

## Content Rules

- **Mental Model** — frameworks for growth decisions, experimentation judgment, analytical reasoning
- **Ship Note** — lessons from tools built, experiments designed, analysis shipped

Every post must come from real experience, be useful to a team member, and require your specific background to write credibly.

## Design

All styling decisions follow `STYLING.md` (Japanese minimalism). Read it before modifying `styles.css` or any layout configuration.
